# 📱 QR Code Generator

A simple and responsive QR Code Generator built using HTML, CSS, and JavaScript. Enter any text or URL, select a size, and instantly generate and download your custom QR code.

## 🚀 Features

- ✅ Real-time QR code generation  
- ✅ Download QR code as PNG image  
- ✅ Multiple size options (100x100 to 1000x1000)  
- ✅ Clean, responsive user interface  
- ✅ Custom styling using CSS variables  

---

## 🛠️ Tech Stack

- **HTML** – Structure of the application  
- **CSS** – Custom styling using variables and media queries  
- **JavaScript** – DOM manipulation, event handling  
- **[qrcodejs](https://github.com/sandeepleel/js-projects)** – Library used for generating QR codes

## 🎯 How to Use

1. Clone this repository  
   ```
   git clone https://github.com/sandeepleel/
   ```

2. Open `index.html` in your browser

3. Type your text or URL into the input field

4. Select the desired size

5. Click **Generate** to create the QR code

6. Click **Download** to save the QR code as an image
