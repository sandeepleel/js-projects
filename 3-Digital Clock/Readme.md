# Digital Clock ⏰

A simple JavaScript-based digital clock that updates in real time.

### 🔹 Features:
- Displays the current time (HH:MM:SS AM/PM).
- Updates every second using `setInterval()`.

### 🛠 Technologies Used:
- HTML
- CSS
- JavaScript

📌 **How to use?**  
Open `index.html` in a browser and see the live clock!
